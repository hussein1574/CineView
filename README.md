## ğŸ› ï¸ Tech Stack & Key Concepts

This project was built to showcase modern Angular practices.

- **Angular (v17+):** Built entirely with **Standalone Components**, directives, and pipes.
- **State Management:** Uses **Signals** as the primary state management tool.
  - **`signal()`:** For all mutable component and service state.
  - **`computed()`:** For deriving state reactively (e.g., `bookmarkedIds`, `currentUserName`).
  - **`input()` / `output()`:** For modern, signal-based component communication.
- **Asynchronous Programming:**
  - **RxJS:** Heavily used for handling asynchronous events and API calls.
  - **Operators:** `switchMap` (for reactive search), `forkJoin` (for AI vibe search), `debounceTime`, `distinctUntilChanged`, `map`, `tap`, and `catchError` are all implemented.
- **Forms:** Uses **Reactive Forms** (`FormControl`) for the search and login pages.
- **Core Features:**
  - **`HttpClient`** configured with **`provideHttpClient`** and **`withInterceptors`**.
  - **Smart `Interceptor` (`apiKeysInterceptor`):** A single interceptor that intelligently adds the correct API key based on the request URL (TMDB or Gemini).
  - **Functional Route Guards:** Uses `CanActivateFn` for `authGuard` and `publicGuard`.
  - **`Custom Pipe`:** A `MinutesToHoursPipe` was created to format movie runtimes.
- **Styling:**
  - **TailwindCSS:** The entire UI is built using Tailwind's utility classes.
  - **Custom Global Styles:** Added global styles for fonts, custom scrollbars, and loaders.
- **Documentation:**
  - Includes a detailed `API_SETUP.md` guide for easy configuration.
  - `.gitignore` is properly configured to exclude sensitive `environment.ts` files.

## ğŸš€ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- [Angular CLI](https://angular.dev/tools/cli) (v17+)

### Installation & Setup

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/hussein1574/CineView.git
    cd CineView
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    ```

3.  **Set up API Keys:**
    This project requires API keys from TMDB and Google Gemini.

    - Follow the detailed instructions in the [**API_SETUP.md**](API_SETUP.md) file to get your keys and create your `environment.ts` file.

4.  **Run the development server:**

    ```bash
    npm start
    ```

    Navigate to `http://localhost:4200/`.
